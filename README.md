Anganwadi Management System  
A complete web-based application built using **Spring Boot**, **JSP**, and **MySQL** to digitalize daily operations of Anganwadis in Kerala.  
This system helps manage child records, attendance, nutrition details, growth tracking, and staff activities efficiently.

Child Management  
- Add, update, delete child records  
- Store child's age, DOB, health notes  
- Daily attendance

Pregnant Women Management
- Add, update, delete records

Item Management
-Add update,delete records
-Monthly item distribution

Staff Management
-Add update,delete records
